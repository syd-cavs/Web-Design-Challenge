Summary: Latitude vs. X

The purpose of the project was to look at temperature, cloudiness, humidity, and wind speed in relation to global points related to latitude.

This was accomplished by using OpenWeatherMap API to obtain cities for the data set.

After the datasets were cleaned up and prepared for usage, plots were created using MatplotLib.

These plots are listed under the Visualizations Tab.
